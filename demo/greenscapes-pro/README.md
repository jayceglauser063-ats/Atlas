# GreenScapes Pro — Demo Landscaping Website

A polished, professional one-page website for **GreenScapes Pro**, a fictional Austin-based landscaping company. Built from the ModernWeb Forge template as a sales demo.

## Quick Start

```bash
cd demo/greenscapes-pro
npm install
npm run dev    # starts dev server
npm run build  # production build
```

## Sections
- **Hero** — Full-screen with gradient, animated background, trust badge, and CTAs
- **Services** — 6 landscaping services with hover animations
- **About** — Company story, stats, and feature highlights
- **Testimonials** — 3 client reviews with star ratings
- **Contact** — Form with name, email, phone, message + business info

## Tech
Vite + Tailwind CSS v4 + vanilla JS. Memory-light, fast build.

## Purpose
Sales demo for prospects — shows what a finished ModernWeb Forge site looks like when customized for a landscaping business.
